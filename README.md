# Aviator Game

A professional full-stack Aviator Game using React (frontend) and Node.js/Express (backend).

## Structure

- `backend/` — Node.js/Express API
- `frontend/` — React app

## Getting Started

1. **Backend**
   ```
   cd backend
   npm install
   npm start
   ```

2. **Frontend**
   ```
   cd frontend
   npm install
   npm start
   ```
